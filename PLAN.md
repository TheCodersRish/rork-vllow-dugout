# Vllow Dugout — AI Cricket Coaching App

## Features

**Feed Tab**
- Morning motivational message from the AI coach with a pulsing green "Live" indicator
- Skill tip cards with large hero images, category badges (Masterclass), bookmark buttons, and "Watch Drill" action
- Daily mission cards showing drill details (target, time, difficulty) with V-Coin rewards and a "Start Tracking Session" button
- Stats glimpse cards showing win streak and global rank with icons and bold numbers

**AI Coach Tab**
- Real-time AI chat powered by the Rork Toolkit for personalized cricket coaching
- Chat bubbles styled differently for the AI coach (dark cards) and user (green accent cards)
- Inline drill video thumbnails within coach responses with play buttons
- Text input bar with a microphone button at the bottom
- Coaching context header showing the current focus area

**Intel Tab**
- Hero performance summary showing latest batting stats (runs, balls, strike rate)
- V-Coin bonus display for match achievements
- Run scoring zone heat map visualization with tactical zone labels
- Dismissal pattern breakdown with a circular chart
- AI coach insight strip with personalized performance analysis

**Arena Tab**
- Global leaderboard with player rankings, avatars, and V-Coin totals
- Filter tabs for different time periods (Weekly, Monthly, All-Time)
- Your rank highlighted with a distinct style
- Challenge buttons to compete with nearby-ranked players

**Store Tab**
- Goal progress tracker showing how close you are to redeeming an item
- Category filter chips (All Gear, Bats, Gloves, Protective, Apparel)
- Product cards with images, names, and V-Coin prices
- Product detail view with specs, power rating, sweet spot visualization
- Promotional banners for limited drops and double-coin events

**Core Systems**
- V-Coins currency system tracked with local persistence
- Bookmarking drills and saving progress locally
- User profile with avatar at the top of every screen

## Design

- **Dark premium theme** with near-black backgrounds and subtle card surfaces
- **Neon green (#39FF14)** as the signature accent color for highlights, indicators, and active states
- Clean, bold typography using the system font with heavy/extrabold weights for headlines
- Uppercase tracking-wide labels for categories and metadata
- Soft shadows and subtle borders on cards for depth
- Floating pill-shaped tab bar at the bottom with a frosted glass effect
- Green pulsing dot for "live" indicators
- Hero image cards with gradient overlays and text on top
- Haptic feedback on key interactions (completing drills, earning coins)

---

## Screens

- **Feed** — Scrollable vertical feed with coach message, skill tip hero card, daily mission card, and stat cards
- **AI Coach** — Chat interface with AI messages, user replies, inline media, and a floating input bar
- **Intel** — Performance dashboard with stat hero, heat map, dismissal chart, and AI insights
- **Arena** — Leaderboard screen with rank list and time-period filters
- **Store** — Product catalog with goal tracker, filter chips, product grid, and promotional sections
- **Product Detail** — Full product page with hero image, specs grid, and redeem button (presented as a pushed screen from Store)
